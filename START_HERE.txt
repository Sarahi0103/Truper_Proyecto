🚀 INSTRUCCIONES DE INICIO RÁPIDO:
───────────────────────────────────────────────────────────────────

PASO 1: Verificar Instalación
   $ bash verify_tickets_system.sh
   
   Esperado: "✅ VERIFICACIÓN COMPLETADA EXITOSAMENTE"

PASO 2: Inicializar Base de Datos
   Opción A (Recomendado - Navegador):
      1. Abre: http://localhost/public/migrate.php
      2. Deberías ver: "✅ Sistema de tickets inicializado correctamente"
      3. Verifica que todas las tablas se crearon
   
   Opción B (Terminal):
      $ php public/migrate.php

PASO 3: Acceder al Panel
   1. Asegurate de estar logueado como ADMIN
   2. Ve a: http://localhost/public/tickets.php
   3. Deberías ver:
      • 📊 Estadísticas (tickets, ventas, devoluciones)
      • 🔍 Filtros de búsqueda
      • 📋 Tabla de tickets
      • ➕ Botón para crear ticket

PASO 4: Crear Primer Ticket
   1. Click en "➕ Crear Ticket"
   2. Llena los datos:
      • Cliente: 1 (admin user)
      • Tipo: Venta
      • Subtotal: 1000.00
      • Impuesto: 190.00
      • Total: 1190.00
   3. Click "Crear Ticket"
   4. Deberías ver el folio generado (ej: 202604-00001)

PASO 5: Verificar en API
   $ curl http://localhost/api/tickets.php?action=list
   
   Deberías ver tu ticket creado en formato JSON
